## About this site :
This is a responsive mobile first single page application using HTML and TailwindCSS compatible for small, medium, large and extra large screens.

## Concepts used to style the page :
- Utility first APIs
- Flexbox
- Positioning
- Backgrounds
- Margins and Padding
- Box Shadows
- Media Queries
